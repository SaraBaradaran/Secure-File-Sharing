# Secure-File-Sharing

## Usage guide

### Requirements
* Python 3.4+

### Step 1: Download iPython Files
You can clone codes using the below command:
```
git clone https://github.com/SaraBaradaran/Secure-File-Sharing
```

### Step 2: Open Jupyter Notebook
```
jupyter notebook
```

Finally, run `./script.sh` file.
